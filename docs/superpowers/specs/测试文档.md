# Voice Industrial Design 测试文档

日期：2026-06-14

## 1. 测试目标

验证 Voice Industrial Design 项目是否能够围绕“工业设计语音脑暴画布”的 MVP 功能完成核心闭环：

1. 用户创建工业设计脑暴会话。
2. 用户通过语音或转写文本输入产品设计需求。
3. 系统识别用户意图，并生成树状概念分支。
4. 系统支持高风险操作的复述确认与取消。
5. 系统支持沿节点继续下钻、刷新当前层、扩充分支和单次撤销。
6. 前端工作台能够正确展示画布、节点、对话、录音状态、当前目标、意图状态和确认卡片。
7. 后端 API、共享 schema、agent 网关和持久层行为符合项目约定。

## 2. 测试范围

### 2.1 覆盖范围

1. Web 工作台页面。
2. 静态预览页面。
3. Session API。
4. Voice turn API。
5. Task 查询、确认、取消 API。
6. Undo API。
7. Orchestrator 任务状态机。
8. 树节点写入、刷新、下钻和撤销。
9. SiliconFlow agent 网关调用。
10. Shared schema 与常量约束。

### 2.2 不覆盖范围

1. 多人协作。
2. CAD、建模或生产链路集成。
3. 高保真工业设计渲染。
4. 鼠标驱动的复杂白板编辑。
5. 键盘文字输入作为主要创作路径。
6. 视觉 review 主链路。

## 11. 验收标准

1. 用户可以创建 session，并看到空白工作台。
2. 用户首轮输入后，系统能生成 3 到 4 个工业设计概念方向。
3. 每个节点都有稳定名称、数字序号、层内序号、草图和解释。
4. `expand_branches` 可直接执行并追加节点。
5. `refresh_layer` 必须先确认，确认后替换当前层，取消后不写树。
6. `branch_deeper` 必须先确认，确认后在目标节点下生成子层。
7. `undo` 能恢复最近一次已确认树操作；无可撤销操作时返回明确错误。
8. 对话区能展示转写、理解、确认、生成和完成摘要。
9. agent 网关能正确处理 ASR、结构化脑暴、草图生成、重试和非法响应。
10. 自动化测试命令可在本地稳定运行。
